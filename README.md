# 📚 JavaScript 코딩테스트 학습 가이드

자바스크립트를 처음 시작하는 분들을 위한 코딩테스트 학습 자료입니다.

## 🎯 학습 순서

### 🌱 **초급 (00 시리즈) - 자바스크립트 기초**

자바스크립트를 처음 접하는 분들을 위한 기초 과정입니다.

| 파일                        | 주제             | 설명                                 |
| --------------------------- | ---------------- | ------------------------------------ |
| `00-1_variables.html`       | 변수와 자료형    | let, const, 다양한 자료형            |
| `00-2_operators.html`       | 연산자           | 산술, 비교, 논리 연산자              |
| `00-3_if-else.html`         | 조건문           | if/else, 논리 연산자, 삼항 연산자    |
| `00-4_loop-for.html`        | for 반복문       | 기본 반복, 배열 순회, break/continue |
| `00-5_loop-while.html`      | while 반복문     | while, do-while, 조건부 반복         |
| `00-6_array-basics.html`    | 배열 기초        | 배열 생성, 접근, 메서드              |
| `00-7_string-basics.html`   | 문자열 기초      | 문자열 조작, 메서드                  |
| `00-8_function-basics.html` | 함수 기초        | 함수 선언, 매개변수, 반환값          |
| `00-9_find-max-min.html`    | 최댓값/최솟값    | 배열에서 극값 찾기                   |
| `00-10_sum-array.html`      | 배열 합계와 평균 | 합계, 평균, 조건부 합계              |

## 📖 학습 방법

### 1. **파일 열기**

각 HTML 파일을 브라우저에서 열어주세요.

```bash
# 방법 1: 직접 더블클릭
파일 탐색기에서 HTML 파일을 더블클릭

# 방법 2: Live Server (VS Code)
VS Code에서 HTML 파일 우클릭 > "Open with Live Server"
```

### 2. **콘솔 확인**

브라우저에서 `F12` (또는 `Cmd+Option+I`)를 눌러 개발자 도구를 열고, Console 탭을 확인하세요.

### 3. **코드 분석**

각 파일의 주석을 꼼꼼히 읽으며 코드의 동작 원리를 이해하세요.

### 4. **직접 수정해보기**

- 변수 값을 바꿔보기
- 함수에 다른 인자 전달해보기
- 새로운 예제 직접 작성해보기

### 5. **실습 문제 풀기**

각 파일 마지막에 있는 "실습 문제"를 먼저 혼자 풀어보고, 답을 확인하세요.

## 💡 학습 팁

### ✅ **초보자를 위한 조언**

1. **순서대로 학습하세요**

   - 00 시리즈를 순서대로 완료한 후 05 시리즈로 넘어가세요.
   - 각 개념이 다음 개념의 기초가 됩니다.

2. **직접 타이핑하세요**

   - 복사/붙여넣기보다 직접 타이핑하면서 익히세요.
   - 손으로 쓰면 기억에 오래 남습니다.

3. **작은 것부터 시작하세요**

   - 한 번에 너무 많은 것을 배우려 하지 마세요.
   - 하루에 1-2개 파일씩 천천히 학습하세요.

4. **에러를 두려워하지 마세요**
   - 에러는 학습의 기회입니다.
   - 콘솔에 나오는 에러 메시지를 읽어보세요.

### 🎓 **효과적인 학습법**

1. **반복 학습**

   - 이해가 안 되는 부분은 여러 번 반복하세요.
   - 다음 날 다시 복습하면 더 잘 이해됩니다.

2. **메모하기**

   - 중요한 개념은 별도로 노트에 정리하세요.
   - 자신만의 치트시트를 만들어보세요.

3. **응용하기**

   - 배운 내용을 실생활 문제에 적용해보세요.
   - 예: 성적 계산기, 할인가 계산기 등

4. **질문하기**
   - 이해가 안 되는 부분은 적극적으로 질문하세요.
   - 스터디 그룹을 만들어 함께 공부하세요.
